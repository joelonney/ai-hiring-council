# AI Hiring Council

AI-powered hiring decision support system that simulates a hiring panel using multiple specialized AI agents.

## Problem

Hiring decisions are often fragmented across resumes, interview notes, and stakeholder opinions. Recruiters spend significant time synthesizing information and decisions can become inconsistent.

## Solution

AI Hiring Council uses:

- Recruiter Agent
- Technical Agent
- Behavioral Agent

Each agent independently evaluates a candidate before an AI Hiring Council generates a final recommendation.

## Features

- Resume PDF Upload
- Job Description Analysis
- Interview Notes Analysis
- Multi-Agent Evaluation
- Explainable Recommendations
- Final Hiring Council Decision
